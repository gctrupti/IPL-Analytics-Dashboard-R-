# IPL Analytics Dashboard

An interactive web dashboard built using R Shiny to analyze IPL data and predict match outcomes.

---

## Features

* Team Performance Analysis
  View match wins and trends for different teams

* Match Winner Prediction
  Machine learning model to predict match results

* Player Statistics
  Explore runs, strike rate, and performance

* Orange Cap Leaderboard
  Top run scorers by season

* Player Comparison
  Compare two players based on key statistics

---

## Tech Stack

* R
* Shiny
* ggplot2
* dplyr
* randomForest

---

## How to Run

1. Clone the repository
2. Open the project in RStudio
3. Run the following command:

```r
shiny::runApp()
```
